# 📊 Ratio and Proportion

## 📌 Introduction
Ratio and Proportion is an important topic in quantitative aptitude, commonly asked in placement exams ,Competitive Exams.

---

## 📖 Basic Concepts

### 🔹 Ratio
A ratio is a comparison of two quantities of the same kind.

Ratio = a/b = a : b

Example:  
10 : 20 = 1 : 2

---

### 🔹 Proportion
A proportion states that two ratios are equal.

a : b = c : d  
or  
a/b = c/d

---

## ⚡ Important Formulas

### 1. Simplification of Ratio
Divide both terms by their HCF  

Example:  
20 : 30 = 2 : 3  

---

### 2. Fourth Proportional
If a : b = c : d  

d = (b × c) / a  

---

### 3. Third Proportional
If a : b = b : c  

c = (b²) / a  

---

### 4. Mean Proportional
If a : x = x : b  

x = √(a × b)  

---

### 5. Duplicate Ratio
(a : b)² = a² : b²  

---

### 6. Sub-duplicate Ratio
√a : √b  

---

## 🔥 Properties of Proportion

If a : b = c : d  

- a × d = b × c  
- Invertendo → b : a = d : c  
- Alternendo → a : c = b : d  
- Componendo → (a + b) : b = (c + d) : d  

---

## 🧠 Short Tricks

If a : b = m : n  

a = km  
b = kn  

---

If total is given:

a + b = Total  

a = (m / (m + n)) × Total  
b = (n / (m + n)) × Total
