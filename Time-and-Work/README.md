## ⚙️ Key Formulas

### 1. Work Formula

Work = Time × Efficiency

### 2. Efficiency

Efficiency = Work / Time

### 3. Time Taken

Time = Work / Efficiency

---

### 4. Combined Work

If A can complete a work in X days,
and B can complete it in Y days:

(A + B)’s 1 day work = (1/X + 1/Y)

---

### 5. Pipes and Cisterns

* Inlet → Positive Work
* Outlet → Negative Work

Net Work = Inlet − Outlet

---

## 🚀 Shortcut Tricks

* If A takes **x days**, then A’s 1-day work = **1/x**
* Always assume total work as **LCM of given times** for easy calculation
* Use ratios when efficiencies are compared
